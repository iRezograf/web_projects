Доступ к странице со справочной информацией по любой Git-команде:

  • $ git help <команда>
  • $ git <команда> --help
  • $ man git-<команда>

Далее:

  • проверяем установлен ли гит:
    • git --version
      • или
    • git status
      • git version 2.32.1 (Apple Git-133)
  • смотрим настройки::
    • git config -l
  • возможно придется проинициализировать некоторые параметры:
    • $ git config --global user.name "Rafael Rezyapov"
    • $ git config --global user.email "Rezograf@gmail.com"
    • git config -l
    • ...
    • user.email=rezograf@gmail.com
    • user.name=Rafael Rezyapov
    • ...
  • базовый набор сохранения изменений:
    • `git status`
    • `git diff`
    • `git add Essey-Summary`
    • `git commit -m "comment about commit"`
    • `git push`-отправить изменения на сервер
    • `git branch NewBranch`-создать новую ветку
    • `git checkout NewBranch`-сделать ветку актуальной
    • `git commit -m "New branch for new lesson"`-коммитим
    • `git commit --amend -m "Новое имя для последнего коммита"`-можно даже переименовать последний коммит
    • `git clone https://github.com/iRezograf/GeekBrains`- создать локальный клон. Он создасться в том же каталоге
